# lau-contact-agents

**Contact geometry for agents: contact manifolds, Reeb fields, contactomorphisms, Legendrian submanifolds, and thermodynamic agent dynamics.**

A Rust library implementing contact geometry — the odd-dimensional cousin of symplectic geometry — with a focus on thermodynamic agent dynamics. Contact geometry provides the natural mathematical framework for dissipative systems, irreversible thermodynamics, and entropy production.

## What This Does

This library provides 9 modules covering contact geometry from foundations to applications:

1. **Contact forms** — The 1-form α on a (2n+1)-manifold satisfying α ∧ (dα)ⁿ ≠ 0, with the associated Reeb vector field.
2. **Contactomorphisms** — Diffeomorphisms preserving the contact structure (φ*α = f·α), with composition and inversion.
3. **Legendrian submanifolds** — n-dimensional submanifolds tangent to the contact distribution (α|_L = 0), including Legendrian knots.
4. **Contact Hamiltonians** — Hamiltonian dynamics on contact manifolds, including dissipative systems where energy is not conserved.
5. **Darboux theorem** — Local normal form: every contact form looks like dz − Σ yᵢ dxᵢ in suitable coordinates.
6. **Gray stability** — Isotopy stability of contact structures: nearby contact forms are diffeomorphic.
7. **Weinstein conjecture** — Every Reeb flow on a closed contact manifold has at least one closed orbit.
8. **Contact homology** — Floer-theoretic invariant generated by Reeb orbits, with differential counting pseudoholomorphic curves.
9. **Thermodynamic dynamics** — Agents on contact manifolds where x = extensive, y = intensive, z = entropy, evolving via contact Hamiltonian flow.

## Key Idea

Contact geometry lives on odd-dimensional manifolds of dimension 2n+1. The defining structure is a 1-form α such that α ∧ (dα)ⁿ ≠ 0 everywhere. This means dα is a symplectic form on the 2n-dimensional contact distribution ξ = ker(α).

Unlike symplectic geometry (which conserves energy), contact geometry naturally models **dissipation**: the contact Hamiltonian H is not conserved along its own flow. The dissipation rate is dH/dt = −R(H)·H where R is the Reeb field.

This makes contact geometry the natural language for thermodynamics:
- Position x = extensive quantities (volume, particle number)
- Momentum y = intensive quantities (pressure, chemical potential)
- z = entropy production (the additional variable unique to contact geometry)

## Quick Start

### Contact Forms and Reeb Fields

```rust
use lau_contact_agents::contact_form::{ContactForm, ReebVectorField};
use nalgebra::DVector;

// Standard contact form on R³: α = dz - y dx
let form = ContactForm::standard(1);
let p = DVector::from_vec(vec![1.0, 2.0, 3.0]);

// Evaluate α at p
let alpha = form.evaluate_at(&p);
// α = (-y, 0, 1) = (-2, 0, 1)

// Check contact condition: α ∧ (dα)ⁿ ≠ 0
assert!(form.is_contact_at(&p));

// The Reeb vector field: R = ∂/∂z
let reeb = ReebVectorField::new(1);
let r = reeb.evaluate(&p);
// Verify: α(R) = 1, dα(R, ·) = 0
assert!(reeb.verify_alpha_condition(&p, &form));
assert!(reeb.verify_dalpha_condition(&p, &form));
```

### Contactomorphisms

```rust
use lau_contact_agents::contactomorphism::Contactomorphism;
use nalgebra::{DVector, DMatrix};

// Identity map (strictly contact)
let id = Contactomorphism::identity(1);

// Rotation in (x,y) plane preserves contact structure
let rot = Contactomorphism::xy_rotation(1, 0, std::f64::consts::PI / 4.0);
assert!(rot.preserves_contact_structure(&DVector::from_vec(vec![0.0, 0.0, 1.0])));

// Compose and invert
let composed = rot.compose(&rot);
let inverse = rot.inverse().unwrap();
let product = rot.compose(&inverse); // ≈ identity
```

### Legendrian Submanifolds

```rust
use lau_contact_agents::legendrian::{LegendrianSubmanifold, LegendrianKnot};
use nalgebra::DVector;

// Standard Legendrian: y=0, z=0 in R³ (the x-axis)
let l = LegendrianSubmanifold::standard(1);
let alpha = DVector::from_vec(vec![0.0, 0.0, 1.0]); // α at y=0
assert!(l.is_legendrian(&alpha));

// From a generating function q(x) = x
let grad_q = DVector::from_vec(vec![1.0]);
let l2 = LegendrianSubmanifold::from_generating_function(1, &grad_q, 1.0);

// Legendrian knot (unknot)
let knot = LegendrianKnot::unknot(100);
assert_eq!(knot.thurston_bennequin(), -1);
```

### Contact Hamiltonian Dynamics

```rust
use lau_contact_agents::contact_hamiltonian::{ContactHamiltonian, DissipativeContactSystem};
use nalgebra::DVector;

// Contact Hamiltonian H = 5 with gradient (1,2,3)
let h = ContactHamiltonian::new(1, 5.0, DVector::from_vec(vec![1.0, 2.0, 3.0]));
let xh = h.vector_field();
// ẋ = ∂H/∂y = 2, ẏ = -∂H/∂x = -1

// Dissipation rate: dH/dt = -R(H)·H
let rate = h.dissipation_rate(3.0); // -15

// Dissipative system: H_diss = ½ γ y²
let gamma = DVector::from_vec(vec![1.0]);
let sys = DissipativeContactSystem::new(1, gamma);
let state = DVector::from_vec(vec![0.0, 1.0, 0.0]);
let new_state = sys.step(&state, 0.1);
```

### Darboux Charts

```rust
// Darboux chart: local coordinates where α = dz - y dx

let chart2 = construct_darboux_chart(1, DVector::from_vec(vec![0.0; 3]), &alpha, &dalpha);
```

### Gray Stability

```rust
use lau_contact_agents::gray_stability::GrayStability;
use nalgebra::{DVector, DMatrix};

let alpha = DVector::from_vec(vec![0.0, 0.0, 1.0]);
let mut dalpha = DMatrix::zeros(3, 3);
dalpha[(0, 1)] = 1.0; dalpha[(1, 0)] = -1.0;

// Check if a path αₜ = (1-t)α₀ + tα₁ stays contact
let result = GrayStability::check_path(1, &alpha, &alpha, &dalpha, &dalpha, 10);
assert!(result.is_isotopic); // identical forms → stable

// Verify for standard forms
assert!(GrayStability::verify(2));
```

### Contact Homology

```rust
use lau_contact_agents::contact_homology::*;

// Generators: Reeb orbits with CZ grading
let g0 = HomologyGenerator::new(0, 0, 1.0);
let g1 = HomologyGenerator::new(1, 1, 2.0);

// Differential: counts J-holomorphic curves (mod 2)

```

### Thermodynamic Agents

```rust
use lau_contact_agents::thermo_dynamics::{ThermoAgent, ThermoMultiAgent};
use nalgebra::DVector;

// Single agent: state = (x, y, z) = (extensive, intensive, entropy)
let mut agent = ThermoAgent::new(1, DVector::from_vec(vec![0.0, 1.0, 0.0]));
let gamma = DVector::from_vec(vec![1.0]);

agent.evolve(0.1, &gamma);
let energy = agent.internal_energy();
let free = agent.free_energy(1.0);
let entropy_prod = agent.entropy_production(&gamma);

// Multi-agent system with coupling
let a1 = ThermoAgent::new(1, DVector::from_vec(vec![0.0, 1.0, 0.0]));
let a2 = ThermoAgent::new(1, DVector::from_vec(vec![1.0, 0.0, 0.0]));
let mut sys = ThermoMultiAgent::decoupled(vec![a1, a2]);
sys.evolve(0.1, &gamma);
```

## How It Works

### Contact Structure

A contact form α on a (2n+1)-manifold M satisfies α ∧ (dα)ⁿ ≠ 0. This means:
- α is a 1-form (covector field)
- dα is a 2-form that is symplectic on ker(α)
- The top wedge α ∧ (dα)ⁿ is a volume form — the "contact condition"

The library represents the standard form α = dz − Σ yᵢ dxᵢ in coordinates (x₁,...,xₙ, y₁,...,yₙ, z).

### Reeb Vector Field

The Reeb field R is uniquely determined by α(R) = 1 and dα(R, ·) = 0. For the standard form, R = ∂/∂z. The Reeb flow generates the "trivial" dynamics — no dissipation, pure translation in z.

### Contact Hamiltonian Flow

Given H: M → ℝ, the contact Hamiltonian vector field X_H satisfies:
- α(X_H) = −H
- dα(X_H, ·) = dH − R(H)α

In standard coordinates: ẋᵢ = ∂H/∂yᵢ, ẏᵢ = −∂H/∂xᵢ + yᵢ∂H/∂z, ż = −H + Σ yᵢ∂H/∂yᵢ.

Unlike symplectic Hamiltonians, contact Hamiltonians are generally **not conserved**: dH/dt = −R(H)·H. This dissipation is the key physical feature.

### Darboux Theorem

Every contact form is locally equivalent to the standard form. The library constructs Darboux charts by finding symplectic bases for dα restricted to the contact distribution, then building the coordinate transformation.

### Gray Stability

If two contact structures can be connected by a smooth path of contact forms (i.e., αₜ ∧ (dαₜ)ⁿ ≠ 0 for all t), then they are isotopic. The library checks this by linearly interpolating and verifying the contact condition at each step.

### Weinstein Conjecture

On a closed contact manifold, the Reeb flow has at least one closed orbit. The library generates standard Reeb orbits on spheres (Hopf fibration for S³) and verifies the conjecture.

### Contact Homology

Chain complex generated by Reeb orbits, graded by Conley-Zehnder index. The differential counts rigid J-holomorphic curves in the symplectization. Satisfies ∂² = 0, giving a topological invariant of the contact structure. The library implements this with mod 2 coefficients.

### Thermodynamic Agents

Agents are points on a contact manifold where:
- x = extensive variables (volume, particle number)
- y = intensive variables (pressure, chemical potential)
- z = entropy production
- Contact Hamiltonian flow gives dissipative equations of motion
- Energy dissipation rate equals entropy production (Second Law)

## The Math

### Contact Form

A 1-form α on M^{2n+1} is **contact** if α ∧ (dα)ⁿ ≠ 0 everywhere. For the standard form on ℝ^{2n+1}:

$$\alpha = dz - \sum_{i=1}^{n} y_i \, dx_i$$

with dα = Σ dxᵢ ∧ dyᵢ.

### Reeb Field

The unique vector field R satisfying α(R) = 1 and ι_R dα = 0. For the standard form, R = ∂/∂z.

### Contactomorphism

A diffeomorphism φ with φ*α = f·α for some positive function f. **Strictly contact** if f ≡ 1. The Jacobian must preserve the contact covector up to scaling.

### Legendrian Submanifold

An n-dimensional submanifold L with α|_L = 0. Equivalently, TL ⊂ ker(α). Example: the x-axis in ℝ³ with α = dz − y dx.

### Contact Hamiltonian Equations

$$\dot{x}_i = \frac{\partial H}{\partial y_i}, \quad \dot{y}_i = -\frac{\partial H}{\partial x_i} + y_i \frac{\partial H}{\partial z}, \quad \dot{z} = -H + \sum y_i \frac{\partial H}{\partial y_i}$$

Dissipation: $\frac{dH}{dt} = -R(H) \cdot H$

### Darboux Theorem

For any contact form α at point p, there exist local coordinates (x, y, z) with α = dz − Σ yᵢ dxᵢ near p.

### Gray Stability

If αₜ is a smooth family of contact forms on a closed manifold (0 ≤ t ≤ 1), then there exists an isotopy φₜ with φₜ*α₀ = fₜ · αₜ.

### Weinstein Conjecture (proved by Taubes, 2007, for dim 3)

On a closed 3-dimensional contact manifold, the Reeb vector field has at least one closed orbit.

### Contact Homology

Chain complex $HC_*(M, α) = \bigoplus_{\gamma} \mathbb{Z}/2$ over good Reeb orbits γ, graded by CZ index. The differential ∂ counts rigid J-holomorphic cylinders. Satisfies ∂² = 0.

### Thermodynamic Contact Geometry

The contact manifold (x, y, z) with H = ½ Σ γᵢ yᵢ² gives:
- ẋᵢ = γᵢ yᵢ (Fourier's law: heat flux ∝ temperature gradient)
- ẏᵢ = −2cᵢxᵢ (restoring force)
- ż = entropy production = Σ γᵢ yᵢ² ≥ 0 (Second Law)
